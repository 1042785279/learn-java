# BearDemo

>初识JavaWeb

关键词：HTML表单、Servlet、tomcat、JSP、MVC思想

- 例子完成的功能：
  - 从前台使用POST获取表单数据 
  - 后台使用BeerSelect类处理数据
  - 最开始使用out直接print html页面，后面使用JSP生成动态页面
- 从这个例子中初次使用了MVC思想：
  - model：BeerSelect后台处理类
  - view：JSP动态页面
  - control：servlet类
